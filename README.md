Robode: A Robot Simulator to learn code
=========================================

Learn code by playing with **Robode**!  :computer:

[Descubre](descubre.inf.um.es) (spanish) is a platform that look for set up computer science and code in high schools.

A tool to code, a bundle of tutorials and challenges and a social network are joined in just one place. In addition, it offers teachers some tools to follow the student improvement.

In this project, I will add a new module to *Descubre*, simulating a robot with 2 wheels, called **Robode**, and its behavior on circuits. **Robode** has proximity sensors and following lines sensors. My goal is make a tool to code the Robode behavior and, in this way, learning code by playing.


#### [Versión en español]


¡Aprende a programar jugando con **Robode**!  :computer:

[*Descubre*](descubre.inf.um.es) es una plataforma que pretende introducir secundaria y bachillerato lo que es la informática y la programación.

En un mismo sitio integra una serie de tutoriales y pruebas, una herramienta para programar online y una red social en la que se puede compartir los programas creados. También proporciona herramientas de seguimiento para facilitar la labor de los profesores.

Este proyecto en concreto, pretende añadir un modulo a *Descubre* simulando un robot de dos ruedas, llamado **Robode**, y su comportamiento en circuitos. **Robode** es una robot con sensores de proximidad y seguimiento de lineas. Mi meta es crear una herramienta que permita programar el comportamiento de **Robode** y así, aprender a programar jugando.
